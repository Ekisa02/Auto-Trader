# Auto-Trader

Auto-Trader is a project built with Kotlin. The following instructions will guide you through the installation and usage of the project.

## Installation

1. Ensure you have [JDK 11 or later](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) installed on your machine.
2. Clone the repository:
   ```bash
   git clone https://github.com/Ekisa02/Auto-Trader.git
   ```
3. Navigate to the project directory:
   ```bash
   cd Auto-Trader
   ```
4. Use a build tool like [Gradle](https://gradle.org/install/) to build the project:
   ```bash
   ./gradlew build
   ```

## Usage

1. To run the project, execute:
   ```bash
   ./gradlew run
   ```
2. Follow the prompts to configure the trading settings according to your preferences.
